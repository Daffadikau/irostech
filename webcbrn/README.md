# webcbrn

Flutter web recreation of the CBRN Tactical Command System dashboard.

## Features

- Tactical dashboard layout with sidebar navigation and top status bar
- Interactive map with layer switching, pan mode, tracking toggle, and locate button
- Incident markers, zone drawing, and unit trails
- Alert timeline, notifications with sound, and mission playbooks
- Data export (alerts JSON/CSV, units JSON)
- CBRN substances database with search, filter, import, export, and add modal

## Run (Web)

1. Install dependencies
	- flutter pub get
2. Start the app
	- flutter run -d chrome

## Build (Web)

- flutter build web
